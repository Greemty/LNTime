Web app de Esteban DECTOT

Entités :
LightNovel
-> title
-> author
-> summary
Library
-> categories
User
-> username

Cette application a pour objectif de povoir ajouter des lightnovels (romans) à leur inventaire.
Il leur est possible de les classer par Genre (catégorie) pour les retrouver sur la page d'accueil.
Plus spécifiquement, ils peuvent créer des listes personnalisées de lightnovels.
